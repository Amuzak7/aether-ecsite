# AETHER ECサイト プロジェクト

## プロジェクト概要
- ブランド名：AETHER
- ターゲット：25〜35代のメンズ（洗練された大人ストリート）
- テイスト：オーバーサイズのシルエット、ミニマルだけど存在感のあるデザイン
- 配色：黒・ベージュ・ネイビーなどの落ち着いたトーン

## 参考サイト
- https://thedola.com/ を参考に、ミニマルでモダンなデザインを採用
- Heroセクションはthedola.comのようなインパクト重視（大きなビジュアル + シンプルなテキスト）
- 商品ページはシンプルでユーザビリティを重視（サイズ選択や価格が一目でわかるレイアウト）
- 全体のトーンは洗練された大人ストリート（thedola.comの雰囲気）

## 技術スタック
- Next.js 15 (App Router)
- Tailwind CSS + shadcn/ui
- Supabase（在庫管理用）
- Vercel（デプロイ）

## デザインルール
- 洗練されたミニマル
- 余白を多めに
- ホバー効果は控えめ（scale-[1.02]程度）
- テキストはstone-900 / stone-500

## 注意点
- メニューは左端からスライドイン（擦りガラス効果）
- MENUボタンはハンバーガーではなく「menu」と表示
- 画像は /public/images/hero/ と /public/images/products/ に配置
