# AETHER デザインシステム

## カラーパレット
| 用途 | 値 |
|------|-----|
| 背景 | stone-50 / white |
| テキストメイン | stone-900 |
| テキストサブ | stone-500 |
| アクセント | stone-800 |
| ボーダー | stone-200 |

## タイポグラフィ
- フォント：Inter（デフォルト）
- ブランド名：Cormorant Garamond（大きめ・文字間広め）

## コンポーネント
### ボタン
- デフォルト：border border-stone-900 text-stone-900 px-6 py-2 text-xs uppercase tracking-widest
- プライマリ：bg-stone-900 text-white px-6 py-2 text-xs uppercase tracking-widest

### カード
- ホバー：scale-[1.02] transition-transform duration-300 shadow-lg
- 画像：aspect-[3/4] object-cover

## レイアウト
- 余白：px-8 md:px-16 lg:px-24（PC）/ px-6（SP）
- グリッド：grid-cols-4（PC）/ grid-cols-2（SP）
